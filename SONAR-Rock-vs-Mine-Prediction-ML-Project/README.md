# 🔍 SONAR Rock vs Mine Prediction
This project uses a machine learning approach to classify sonar signal data as either rock or mine. The data consists of 60 numeric features representing energy reflected at different frequencies.

## 📌 Project Overview
Dataset: UCI Sonar Dataset-https://archive.ics.uci.edu/dataset/151/connectionist+bench+sonar+mines+vs+rocks

Goal: Build a binary classification model to distinguish between sonar signals bounced off mines and rocks.

## 🧰 Technologies Used
Python

NumPy, Pandas

scikit-learn

seaborn, matplotlib

Prediction

Custom input prediction to classify new sonar readings

## 📈 Results
Achieved accuracy of ~80-85% on test data.

The model reliably predicts sonar target type based on 60 feature readings.
